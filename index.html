What is The Archimedes Appendix?

This website is intended to be a tool for learning about enginerring and simple machines. This website features many examples, some interactive, to engage people. The provided lessons and examples can be used as both a educational and fun resource. Essentially, this website is an appendix for machines and mechanisms.

What are Machines

"Machines are devices used to alter, transmit, and direct forces to accomplish a specific objective." In essence a machine does something to accomplish a goal. A great deminstration of machines is a Rube Goldberg machine. A Rube Goldberg machine, named after American cartoonist Rube Goldberg, is a machine or contraption designed to perform a simple task in an impractical and overly complicated way by adding an unessesacry amount of mechanisms and steps.
